# fishino-button
Progetto per la creazione di un pulsante che accende un led

![alt text](http://www.andreamarzolla.it/github/immagini/fishino-led-pulsante.jpg)

## Necessario
- Fishino UNO (https://fishino.it/fishino-uno-it.html)
- Breadboard
- LED
- Pulsante
- Resistore 220 ohm
- Resistore 100 Kohm
